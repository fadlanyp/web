Apa Itu Html?

HTML, atau HyperText Markup Language, adalah bahasa markup standar yang digunakan untuk membuat dan merancang halaman web. Dikembangkan oleh Tim Berners-Lee pada tahun 1991, HTML merupakan elemen dasar dari World Wide Web. HTML digunakan untuk struktur dasar sebuah halaman web, dan dapat disematkan dengan CSS (Cascading Style Sheets) dan JavaScript untuk memberikan tampilan dan interaktivitas yang lebih kompleks. Berikut adalah beberapa poin utama tentang HTML:

Elemen Dasar

HTML menggunakan elemen-elemen dasar seperti < html>, < head>, < title>, < body>, < p>, < h1>, dan sebagainya untuk membentuk struktur halaman web

Tags:

Setiap elemen HTML dikelilingi oleh tag. Contohnya, tag untuk paragraf adalah < p>, sedangkan tag untuk heading adalah < h1>, < h2>, dan seterusnya.

Atribut:

Elemen HTML dapat memiliki atribut yang memberikan informasi tambahan tentang elemen tersebut. Misalnya, < img> memiliki atribut src untuk menentukan sumber gambar.

Hyperlink:

HTML digunakan untuk membuat hyperlink dengan tag < a>, yang dapat mengarahkan pengguna ke halaman lain atau sumber daya web.

Daftar:

HTML mendukung pembuatan daftar dengan tag < ul> (unordered list) untuk daftar tak terurut dan < ol> (ordered list) untuk daftar terurut.

Formulir

HTML menyediakan tag-form seperti < form>, < input>, < select>, dan lainnya untuk membuat formulir interaktif yang memungkinkan pengguna memberikan input.

HTML terus berkembang dengan penambahan fitur baru untuk mendukung teknologi web yang lebih canggih. Pengembangan HTML saat ini diawasi oleh World Wide Web Consortium (W3C) dan WhatWG
